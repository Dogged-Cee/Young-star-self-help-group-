# ⭐ Young Star Self Help Group (Chama Management System)

A modern web-based **SACCO / Chama management system** built with HTML, CSS, and JavaScript.

It helps manage:
- Member registration
- Chama contributions
- Treasurer approvals
- Financial tracking
- Active members system

---

## 🚀 Features

### 🧾 Member System
- Register members with name, category, and payment
- Two categories:
  - Registration Fee (50 KES)
  - Chama Money (contributions)

### 👨‍💼 Treasurer Dashboard
- Password protected system
- Approve all payments
- Change treasurer password
- View pending approvals

### 👥 Members Management
- Approved members become ACTIVE
- Automatic member ID generation (e.g. YS-001)

### 💰 Chama Tracking
- Tracks all approved contributions
- Displays total chama money
- Receipt generation for each payment

### 📊 Dashboard Analytics
- Active members count
- Pending approvals
- Total chama money
- Pie chart (Registration vs Chama)

### 🎨 UI Features
- Modern dark theme
- Glassmorphism design
- Animated Chama cards
- Responsive layout
- GitHub Pages ready
